# 📌 Project Overview

This project presents an end-to-end Data Analytics solution for Ola ride-booking data using **SQL, Excel, and Power BI**.

The objective is to analyze ride bookings, revenue generation, customer behavior, vehicle performance, cancellation trends, and ratings to uncover actionable business insights and support data-driven decision-making.

The project covers the complete analytics lifecycle:

- Data Cleaning & Preparation
- SQL-Based Analysis
- KPI Development
- Interactive Dashboard Creation
- Business Insights Generation

# ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/yourusername/ola-booking-analytics-dashboard.git
```

### Open Power BI Dashboard

1. Download repository
2. Open `Ola_Dashboard.pbix`
3. Refresh data if required
4. Explore interactive dashboards

### Run SQL Queries

1. Import dataset into MySQL
2. Execute queries from:

```sql
SQL/ola_sql_queries.sql
```
